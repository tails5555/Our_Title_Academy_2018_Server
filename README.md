## Description

소프트웨어 캡스톤 디자인의 후속 프로젝트 진행을 위한 Web Application Server 제작.

## Server Structure

1. User REST API - 사용자 계정, 프로필 관리 서버

2. Context REST API - 제목학원 요청글, 사진, 제목, 반응, 댓글

3. Notice REST API - 공지사항, 관리자 FAQ

## Using Technologies

- 데이터베이스 : MySQL, Redis
